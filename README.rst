====================================
django package of an article
====================================

Anarticle uses tag, catelog, and article models to make article publication easy.
Support for graphQL with pre-defined types and basic resolvers.

------------
Requirements
------------

* Python 3.11+
* django 4.0+
* ariadne 0.16.0+
